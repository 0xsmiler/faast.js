# Contributing to faast.js

## Building

```
$ npm install
$ npm run build
```

### All "fast" live cloud tests

First follow the instructions in the README to setup accounts on cloud
providers.

```
$ npm run test
```

Redirect stdout to get clean output:

```
$ npm run test > out
$ cat out
```

## Local Testing

```
$ npm run test-local
```
